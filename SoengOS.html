About SoengOS

SoengOS is a web-enabled Linux operating system concept inspired by Zen minimalism. It embodies the philosophy of "less is more" — extreme focus, no distractions, and precise engineering aesthetics.

Built as a single-file browser prototype using vanilla JavaScript and CSS. All applications run entirely within the browser. No external dependencies, no frameworks, no runtime.
